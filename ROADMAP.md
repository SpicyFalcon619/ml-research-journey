# Field Guide: Charting a Course into ML & AI Research

A field guide, compiled for one traveler. Follow the path top to bottom. Every numbered basecamp is a stage.

---

## BASECAMP 0: Python, for someone who already codes (1-2 weeks)
*Skip beginner-from-zero courses entirely. You need the syntax differences, the scientific stack, and comfort in notebooks.*

- [x] **The Python Tutorial (docs.python.org)**
  - [x] Variables, Data Types, If/Else, Loops
  - [x] Functions, Lists, Dictionaries, Sets, Tuples, Error Handling
- [x] **NumPy Quickstart**
  - [x] Speed Comparison
  - [x] Array Generators & Matrix Math (Broadcasting)
  - [x] Reshaping, Indexing, Masking, Stacking
  - [x] Random Numbers
- [x] **pandas: Getting Started**
  - [x] Series & DataFrames (Creating tables)
  - [x] Loading Data (Reading CSV/Excel files)
  - [x] Inspecting Data (`head()`, `info()`, `describe()`)
  - [x] Selecting & Filtering Data (`loc`, `iloc`)
  - [x] Handling Missing Data
  - [x] Grouping & Aggregating (`groupby`)

---

## SUPPLY LINE: Math, picked up as you walk (ongoing)
*Not a gate before basecamp 1 — a supply line that runs alongside the whole trail. Learn each piece when a concept ahead needs it.*

- [ ] **Essence of Linear Algebra (3Blue1Brown):** Vectors, matrices, dot products
- [ ] **Essence of Calculus (3Blue1Brown):** Derivatives and gradients
- [ ] **Statistics & Probability (Khan Academy):** Distributions, expectation, Bayes' theorem

---

## BASECAMP 1: Classical machine learning (4-6 weeks)
*Where you learn what ML actually is, before deep learning. Regression, classification, trees, the full real workflow.*

- [ ] **Machine Learning Specialization (Andrew Ng):** Still the gold standard for building real intuition.
- [ ] **scikit-learn Tutorials:** Implement everything from the course above on real data.
- [ ] **Titanic — Machine Learning from Disaster (Kaggle):** The classic first project.

---

## BASECAMP 2: Deep learning (6-8 weeks)
*Where it gets fun. Neural networks, and the framework the field has standardized on — PyTorch.*

- [ ] **Practical Deep Learning for Coders (fast.ai):** Project-first: you'll train and deploy a real model by lesson two.
- [ ] **Deep Learning Specialization (Andrew Ng):** More rigorous, theory-first. CNNs, RNNs, and the math.
- [ ] **Neural Networks: Zero to Hero (Andrej Karpathy):** Builds backprop and a GPT from scratch, in code.

---

## BASECAMP 3: ML Engineering & MLOps (2-4 weeks)
*Taking models out of local notebooks and into production.*

- [ ] **Full Stack Deep Learning (FSDL):** The premier course on what happens after you train a model.
- [ ] **Weights & Biases (W&B):** The industry standard for experiment tracking.

---

## 🔀 THE FORK IN THE TRAIL
*Once the fundamentals click, you'll notice yourself pulled one direction. Pick a branch based on what you want to build.*

- **If you're drawn to images (Computer Vision):** CS231n (Stanford)
- **If you're drawn to language (NLP & LLMs):** CS224n (Stanford) or The LLM Course (Hugging Face)
- **If you're drawn to making things (Generative):** Deep Learning to Stable Diffusion (fast.ai Part 2)
- **If you're drawn to decision making (Reinforcement Learning):** Spinning Up in Deep RL (OpenAI)
- **The ultimate destination (Core Research):** Learn to read papers efficiently (arXiv, Transformers, ResNets)
